ES7 - to do clean async codes*

async, await keyword
#Ref - http://bit.ly/1FkbzcF

Promises are good, but makes code less readable and error prone. With ES7, introduction of async and await keyword - we can write asynchronous code like synchronous blocks

Async via - Promises code

                var db = new PouchDB('WarsawJS')
                db.post({'Speaker':'Sambath'}).then(function(result){
                    db.get(result.id);
                    }).then(function(doc){
                    console.log(doc);
                    }).catch(function(error){
                    console.log(error);
                    });
